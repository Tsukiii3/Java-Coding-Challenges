# Two Sum

## Descrição

Dado um array de inteiros e um valor alvo (`target`), encontre os índices de dois números cuja soma seja igual ao alvo.

## Categoria

- Arrays

## Dificuldade

Easy

## Conceitos

- Arrays
- Loops
- HashMap (solução otimizada)

## Complexidade

### Solução Brute Force

- Tempo: O(n²)
- Espaço: O(1)
