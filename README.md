# Java Coding Challenges

Repositório dedicado à prática de algoritmos, estruturas de dados e resolução de problemas utilizando **Java**.

O objetivo deste projeto é documentar minha evolução como desenvolvedor, praticando desafios clássicos de plataformas como **LeetCode**, **HackerRank** e outros sites de programação.

---

## Objetivos

- Praticar lógica de programação
- Aprimorar conhecimento em Java
- Estudar estruturas de dados
- Resolver problemas de algoritmos
---

## Estrutura do projeto

```text
java-coding-challenges/
│
├── Arrays/
├── Strings/
├── Stack/
├── Sorting/
```

Cada exercício possui sua própria pasta contendo:

- `Solution.java` → solução do problema
- `README.md` → descrição, conceitos e complexidade
