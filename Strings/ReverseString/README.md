# Reverse String

## Descrição

Dado um array de caracteres, inverta sua ordem sem utilizar memória adicional para criar outro array.

### Exemplo

**Entrada**

```java
char[] s = {'h', 'e', 'l', 'l', 'o'};
```

**Saída**

```text
['o', 'l', 'l', 'e', 'h']
```

---

## Categoria

- Strings
- Two Pointers

## Dificuldade

Easy

## Conceitos

- Arrays
- Strings
- Two Pointers
- Troca de elementos

## Solução

A solução utiliza dois ponteiros:

- Um ponteiro inicia no primeiro elemento do array.
- O outro inicia no último elemento.
- Os caracteres são trocados até que os ponteiros se encontrem.

Essa abordagem realiza a inversão diretamente no próprio array, sem utilizar memória extra.

## Complexidade
- Tempo:  **O(n)**
- Espaço:  **O(1)**
